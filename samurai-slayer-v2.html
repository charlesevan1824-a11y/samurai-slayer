USABILITY GUIDE

Principles powering this game's UX design

What is Usability?

Usability refers to how effectively, efficiently, and satisfyingly a user can interact with a system. In Samurai Slayer, every design decision from button placement to feedback animations is guided by three core usability principles.

LEARNABILITY

How easily new users understand the system

  • Predictability Enemies always follow the same arc pattern
  • Familiarity Tap-to-slice gesture mirrors real swipe behavior
  • Consistency Same visual language across all screens
  • Generalizability Skills transfer across difficulty levels
  • Synthesizability Score reflects all past actions clearly

IN-GAME EXAMPLE

The 321 countdown before gameplay gives players a predictable, learnable start ritual every session.

FLEXIBILITY

How many ways users can interact with the system

  • Dialogue Initiative Players choose persona & difficulty
  • Multithreading Pause, shop, and leaderboard run in parallel
  • Task Migrability Switch persona before each new game
  • Substitutivity Tap OR click works on any device
  • Customizability Equip different skins, weapons, clothes

IN-GAME EXAMPLE

Power-up activations (Shield, 2 Points, Slow Time) give players multiple strategic paths during gameplay.

ROBUSTNESS

How well the system supports user goals

  • Observability Live score, combo, lives always visible
  • Recoverability Shield power-up absorbs one fatal hit
  • Responsiveness Instant slash feedback on every tap
  • Task Conformance Game saves stats and coins automatically

IN-GAME EXAMPLE

Red damage flash + screen shake gives instant, observable feedback when a player loses a life supporting error awareness.

Micro-Interactions in This Game

SLASH EFFECT

Animated scissor icon flies off every tapped enemy confirming the hit visually.

COMBO POPUP

Bold animated text appears at enemy position when combo 3 motivating chain kills.

DAMAGE FLASH

Red vignette + screen shake on life loss visceral, immediate feedback.

LEVEL UP BURST

Full-screen gold glow with level text rewarding achievement signal.

Content Strategy

TONE

Intense, action-oriented, empowering. Uses all-caps labels ("START BATTLE", "LEVEL UP!") to drive urgency and excitement.

STYLE

Dark warrior aesthetic with neon accents. Japanese typography adds cultural authenticity. Orbitron font evokes sci-fi precision.

NOTIFICATIONS

In-game only no push notifications. Power-up indicators use pulsing animations to attract attention without disrupting flow.

HOW TO ADD MUSIC & SOUND EFFECTS

METHOD 1 HTML Audio Element

<audio id="bgm" src="music.mp3" loop autoplay></audio>

Place your .mp3/.ogg file in the same folder. Call document.getElementById("bgm").play() on game start.

METHOD 2 Web Audio API (for SFX)

const ctx = new AudioContext();
const buf = await fetch("slash.mp3");
const src = ctx.createBufferSource();
src.connect(ctx.destination); src.start();

Best for short sound effects lower latency than HTML Audio.
